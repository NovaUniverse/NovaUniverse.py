Install/Set Up
==============
1. **Install package from pip.**

.. code-block:: sh

       #Windows/Linux/MacOS
       
       pip install novauniverse

2. **That's It!** 

.. code-block:: python

       import novauniverse as nova

       player = nova.find.player(player_name="THEGOLDENPRO")

       print(player.first_join)

       #OUTPUT: 2021-01-15 19:28:32
